电脑坏了/换新电脑 恢复流程
====================================

前提：新电脑是 Mac（Apple Silicon 芯片）

------------------------------------
第一步：下载 Codex 桌面版
------------------------------------
打开 https://codex.ai 下载安装
安装完成后打开 Codex，登录你的账号

------------------------------------
第二步：下载本项目
------------------------------------
打开终端（Launchpad → 其他 → 终端）
粘贴以下命令回车：

  git clone git@github.com:chengshuaimao9-design/CodexBridge.git ~/Desktop/CodexBridge

这会把项目下载到桌面上的 CodexBridge 文件夹

------------------------------------
第三步：运行安装脚本
------------------------------------
在终端里继续粘贴：

  cd ~/Desktop/CodexBridge && bash setup.sh

脚本会自动安装 Node.js、项目依赖、创建配置
最后会显示一个二维码，用手机微信扫码登录

------------------------------------
第四步：启动桥接
------------------------------------
安装完成后，在终端里粘贴：

  cd ~/Desktop/CodexBridge && bash bridge-runner.sh &

看到 "启动 WeChat bridge" 就说明成功了
现在可以在手机微信上发消息了

------------------------------------
日常使用命令
------------------------------------
启动： cd ~/Desktop/CodexBridge && bash bridge-runner.sh &
停止： pkill -9 -f "node.*cli.ts.*weixin"

微信上的命令：
  停止/中断 — 停止当前处理
  状态      — 查看桥接是否在线
  健康      — 查看完整链路状态
  暂停      — 暂停消息处理
  继续/恢复 — 恢复消息处理
  新对话    — 开启新会话
  帮助      — 显示全部命令

------------------------------------
把本地修改备份到 GitHub
------------------------------------
  cd ~/Desktop/CodexBridge
  git add .
  git commit -m "备注你改了啥"
  git push origin main
